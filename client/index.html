GUÍA DE CONFIGURACIÓN

Conecta el tenant de tu cliente en 5 pasos

BeAudit usa una App Registration de Azure con permisos de solo lectura. El cliente ejecuta un script de PowerShell que lo configura todo en menos de 2 minutos.

1
El cliente descarga el script de PowerShell
Proporciona al cliente el fichero BeAudit-Setup.ps1. Solo necesita una cuenta con rol Global Administrator en el tenant.
2
Ejecuta el script (2 minutos)
El script crea la App Registration, asigna los 8 permisos de lectura y genera el secreto. Sin permisos de escritura.
PowerShell.exe -ExecutionPolicy Bypass -File BeAudit-Setup.ps1
3
El script muestra 3 valores
Al terminar: Tenant ID, Client ID y Client Secret. El cliente los envía a BeServices por canal seguro.
4
BeServices configura las credenciales
Con los 3 valores, actualizas las variables en el servidor BeAudit. Las credenciales nunca viajan al navegador.
5
Lanzas el diagnóstico en la reunión
Abre BeAudit en el navegador con el cliente delante. En 20-30 segundos tienes el dashboard completo con datos reales del tenant.
Alternativa manual
El cliente puede crear la App Registration desde portal.azure.com → Microsoft Entra ID → Registros de aplicaciones
